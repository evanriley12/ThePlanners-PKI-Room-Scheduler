Modernize node.\+js to current ECMAScript specifications! node.\+js will not update their API to ES6+ \href{https://github.com/joyent/node/issues/7549}{\texttt{ for a while}}. This library is a wrapper for various aspects of node.\+js\textquotesingle{} API.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mz_2_r_e_a_d_m_e_autotoc_md16999}{}\doxysection{\texorpdfstring{Installation and Usage}{Installation and Usage}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mz_2_r_e_a_d_m_e_autotoc_md16999}
Set {\ttfamily mz} as a dependency and install it.


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ i\ mz}

\end{DoxyCode}

\DoxyCodeLine{fs.exists(\_\_filename).then(function\ (exists)\ \{}

Many node methods are converted into promises. Any properties that are deprecated or aren\textquotesingle{}t asynchronous will simply be proxied. The modules wrapped are\+:


\begin{DoxyItemize}
\item {\ttfamily child\+\_\+process}
\item {\ttfamily crypto}
\item {\ttfamily dns}
\item {\ttfamily fs} (uses {\ttfamily graceful-\/fs} if available)
\item {\ttfamily readline}
\item {\ttfamily zlib}
\end{DoxyItemize}
