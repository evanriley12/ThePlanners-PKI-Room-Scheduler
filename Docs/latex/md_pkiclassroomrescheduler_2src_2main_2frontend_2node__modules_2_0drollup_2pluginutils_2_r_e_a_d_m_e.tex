Type\+: {\ttfamily String \texorpdfstring{$\vert$}{|} Boolean \texorpdfstring{$\vert$}{|} null}

Optionally resolves the patterns against a directory other than {\ttfamily process.\+cwd()}. If a {\ttfamily String} is specified, then the value will be used as the base directory.
