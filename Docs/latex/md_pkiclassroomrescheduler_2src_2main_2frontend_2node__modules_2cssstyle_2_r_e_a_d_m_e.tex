This package is an extension of the CSSStyle\+Declaration class in Nikita Vasilyev\textquotesingle{}s \href{https://github.com/NV/CSSOM}{\texttt{ CSSOM}} with added support for CSS 2 \& 3 properties. The primary use case is for testing browser code in a Node environment.

It was originally created by Chad Walker, it is now maintaind by Jon Sakas and other open source contributors.

Bug reports and pull requests are welcome.
