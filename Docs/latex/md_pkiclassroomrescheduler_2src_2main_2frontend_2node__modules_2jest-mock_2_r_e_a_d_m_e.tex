Generates a mock based on the given metadata (Metadata for the mock in the schema returned by the get\+Metadata method of this module). Mocks treat functions specially, and all mock functions have additional members, described in the documentation for {\ttfamily fn} in this module.

One important note\+: function prototypes are handled specially by this mocking framework. For functions with prototypes, when called as a constructor, the mock will install mocked function members on the instance. This allows different instances of the same constructor to have different values for its mocks member and its return values.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2jest-mock_2_r_e_a_d_m_e_autotoc_md15418}{}\doxysubsection{\texorpdfstring{{\ttfamily get\+Metadata(component)}}{{\ttfamily get\+Metadata(component)}}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2jest-mock_2_r_e_a_d_m_e_autotoc_md15418}
