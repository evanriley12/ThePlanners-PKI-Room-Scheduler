Enforce that {\ttfamily aria-\/hidden="{}true"{}} is not set on focusable elements.

{\ttfamily aria-\/hidden="{}true"{}} can be used to hide purely decorative content from screen reader users. An element with {\ttfamily aria-\/hidden="{}true"{}} that can also be reached by keyboard can lead to confusion or unexpected behavior for screen reader users.
