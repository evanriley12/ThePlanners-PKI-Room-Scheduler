This default preset for cssnano only includes transforms that make no assumptions about your CSS other than what is passed in. In previous iterations of cssnano, assumptions were made about your CSS which caused output to look different in certain use cases, but not others. These transforms have been moved from the defaults to other presets, to make this preset require only minimal configuration.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8097}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8097}
\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8098}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8098}
Note that this preset comes bundled with cssnano {\itshape by default}, so you don\textquotesingle{}t need to install it separately.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8099}{}\doxysubsection{\texorpdfstring{Configuration}{Configuration}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2cssnano-preset-default_2_r_e_a_d_m_e_autotoc_md8099}
If you would like to use the default configuration, then you don\textquotesingle{}t need to add anything to your {\ttfamily package.\+json}.

But should you wish to customise this, you can pass an array with the second parameter as the options object to use. For example, to remove all comments\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\ \{}
\DoxyCodeLine{\ \ \ "{}name"{}:\ "{}awesome-\/application"{},}
\DoxyCodeLine{+\ \ "{}cssnano"{}:\ \{}
\DoxyCodeLine{+\ \ \ \ "{}preset"{}:\ [}
\DoxyCodeLine{+\ \ \ \ \ \ "{}default"{},}
\DoxyCodeLine{+\ \ \ \ \ \ \{"{}discardComments"{}:\ \{"{}removeAll"{}:\ true\}\}}
\DoxyCodeLine{+\ \ \ \ ]}
\DoxyCodeLine{+\ \ \}}
\DoxyCodeLine{\ \}}

\end{DoxyCode}


Depending on your usage, the JSON configuration might not work for you, such as in cases where you would like to use options with customisable function parameters. For this use case, we recommend a {\ttfamily cssnano.\+config.\+js} at the same location as your {\ttfamily package.\+json}. You can then load a preset and export it with your custom parameters\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ defaultPreset\ =\ require('cssnano-\/preset-\/default');}
\DoxyCodeLine{}
\DoxyCodeLine{module.exports\ =\ defaultPreset(\{}
\DoxyCodeLine{\ \ discardComments:\ \{}
\DoxyCodeLine{\ \ \ \ remove:\ (comment)\ =>\ comment[0]\ ===\ '@',}
\DoxyCodeLine{\ \ \},}
\DoxyCodeLine{\});}

\end{DoxyCode}
