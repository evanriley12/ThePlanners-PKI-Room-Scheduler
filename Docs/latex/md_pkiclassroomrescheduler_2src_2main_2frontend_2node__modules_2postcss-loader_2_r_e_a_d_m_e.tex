 \doxysubparagraph*{Config Files}

The loader will search up the directory tree for configuration in the following places\+:


\begin{DoxyItemize}
\item a {\ttfamily postcss} property in {\ttfamily package.\+json}
\item a {\ttfamily .postcssrc} file in JSON or YAML format
\item a {\ttfamily .postcssrc.\+json}, {\ttfamily .postcssrc.\+yaml}, {\ttfamily .postcssrc.\+yml}, {\ttfamily .postcssrc.\+js}, or {\ttfamily .postcssrc.\+cjs} file
\item a {\ttfamily postcss.\+config.\+js} or {\ttfamily postcss.\+config.\+cjs} Common\+JS module exporting an object ({\bfseries{recommended}})
\end{DoxyItemize}
