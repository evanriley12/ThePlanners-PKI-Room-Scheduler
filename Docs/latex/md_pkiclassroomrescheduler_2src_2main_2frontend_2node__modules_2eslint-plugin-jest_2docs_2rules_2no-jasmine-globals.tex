 {\ttfamily jest} uses {\ttfamily jasmine} as a test runner. A side effect of this is that both a {\ttfamily jasmine} object, and some jasmine-\/specific globals, are exposed to the test environment. Most functionality offered by Jasmine has been ported to Jest, and the Jasmine globals will stop working in the future. Developers should therefore migrate to Jest\textquotesingle{}s documented API instead of relying on the undocumented Jasmine API.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-jest_2docs_2rules_2no-jasmine-globals_autotoc_md10420}{}\doxysubsection{\texorpdfstring{Rule details}{Rule details}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-jest_2docs_2rules_2no-jasmine-globals_autotoc_md10420}
This rule reports on any usage of Jasmine globals which is not ported to Jest, and suggests alternative from Jest\textquotesingle{}s own API.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-jest_2docs_2rules_2no-jasmine-globals_autotoc_md10421}{}\doxysubsection{\texorpdfstring{Default configuration}{Default configuration}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-jest_2docs_2rules_2no-jasmine-globals_autotoc_md10421}
The following patterns are considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{jasmine.DEFAULT\_TIMEOUT\_INTERVAL\ =\ 5000;}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ pending();}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ fail();}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ spyOn(some,\ 'object');}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ jasmine.createSpy();}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect('foo').toEqual(jasmine.anything());}
\DoxyCodeLine{\});}

\end{DoxyCode}


The following patterns would not be considered warnings\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{jest.setTimeout(5000);}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ jest.spyOn(some,\ 'object');}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ jest.fn();}
\DoxyCodeLine{\});}
\DoxyCodeLine{}
\DoxyCodeLine{test('my\ test',\ ()\ =>\ \{}
\DoxyCodeLine{\ \ expect('foo').toEqual(expect.anything());}
\DoxyCodeLine{\});}

\end{DoxyCode}
