\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2common-path-prefix_2_r_e_a_d_m_e_autotoc_md7370}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2common-path-prefix_2_r_e_a_d_m_e_autotoc_md7370}
The module has one default export, the {\ttfamily common\+Path\+Prefix} function\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ commonPathPrefix\ =\ require('common-\/path-\/prefix')}

\end{DoxyCode}


Call {\ttfamily common\+Path\+Prefix()} with an array of paths (strings) and an optional separator character\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{const\ paths\ =\ ['templates/main.handlebars',\ 'templates/\_partial.handlebars']}
\DoxyCodeLine{}
\DoxyCodeLine{commonPathPrefix(paths,\ '/')\ //\ returns\ 'templates/'}

\end{DoxyCode}


If the separator is not provided the first {\ttfamily /} or {\ttfamily \textbackslash{}} found in any of the paths is used. Otherwise the platform-\/default value is used\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{commonPathPrefix(['templates/main.handlebars',\ 'templates/\_partial.handlebars'])\ //\ returns\ 'templates/'}
\DoxyCodeLine{commonPathPrefix(['templates\(\backslash\)\(\backslash\)main.handlebars',\ 'templates\(\backslash\)\(\backslash\)\_partial.handlebars'])\ //\ returns\ 'templates\(\backslash\)\(\backslash\)'}

\end{DoxyCode}


You can provide any separator, for example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{commonPathPrefix(['foo\$bar',\ 'foo\$baz'],\ '\$')\ //\ returns\ 'foo\$''}

\end{DoxyCode}


An empty string is returned if no common prefix exists\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{commonPathPrefix(['foo/bar',\ 'baz/qux'])\ //\ returns\ ''}
\DoxyCodeLine{commonPathPrefix(['foo/bar'])\ //\ returns\ ''}

\end{DoxyCode}


Note that the following {\itshape does} have a common prefix\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{commonPathPrefix(['/foo/bar',\ '/baz/qux'])\ //\ returns\ '/'}

\end{DoxyCode}
