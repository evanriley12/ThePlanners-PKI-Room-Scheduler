This rule reports any imports that come after non-\/import statements.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9866}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9866}

\begin{DoxyCode}{0}
\DoxyCodeLine{import\ foo\ from\ './foo'}
\DoxyCodeLine{}
\DoxyCodeLine{//\ some\ module-\/level\ initializer}
\DoxyCodeLine{initWith(foo)}
\DoxyCodeLine{}
\DoxyCodeLine{import\ bar\ from\ './bar'\ //\ <-\/\ reported}

\end{DoxyCode}


Providing {\ttfamily absolute-\/first} as an option will report any absolute imports (i.\+e. packages) that come after any relative imports\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ foo\ from\ 'foo'}
\DoxyCodeLine{import\ bar\ from\ './bar'}
\DoxyCodeLine{}
\DoxyCodeLine{import\ *\ as\ \_\ from\ 'lodash'\ //\ <-\/\ reported}

\end{DoxyCode}


If you really want import type ordering, check out \doxysectlink{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2order}{{\ttfamily import/order}}{0}.

Notably, {\ttfamily import}s are hoisted, which means the imported modules will be evaluated before any of the statements interspersed between them. Keeping all {\ttfamily import}s together at the top of the file may prevent surprises resulting from this part of the spec.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9867}{}\doxysubsection{\texorpdfstring{On directives}{On directives}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9867}
Directives are allowed as long as they occur strictly before any {\ttfamily import} declarations, as follows\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{'use\ super-\/mega-\/strict'}
\DoxyCodeLine{}
\DoxyCodeLine{import\ \{\ suchFoo\ \}\ from\ 'lame-\/fake-\/module-\/name'\ \ //\ no\ report\ here}

\end{DoxyCode}


A directive in this case is assumed to be a single statement that contains only a literal string-\/valued expression.

`\textquotesingle{}use strict'{\ttfamily would be a good example, except that \mbox{[}modules are always in strict mode\mbox{]}(\href{https://262.ecma-international.org/6.0/}{\texttt{ https\+://262.\+ecma-\/international.\+org/6.\+0/}}\#sec-\/strict-\/mode-\/code) so it would be surprising to see a}\textquotesingle{}use strict\textquotesingle{}{\ttfamily sharing a file with}import{\ttfamily s and }export\`{}s.

Given that, see \href{https://github.com/import-js/eslint-plugin-import/issues/255}{\texttt{ \#255}} for the reasoning.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9868}{}\doxysubsection{\texorpdfstring{With Fixer}{With Fixer}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9868}
This rule contains a fixer to reorder in-\/body import to top, the following criteria applied\+:


\begin{DoxyEnumerate}
\item Never re-\/order relative to each other, even if {\ttfamily absolute-\/first} is set.
\item If an import creates an identifier, and that identifier is referenced at module level {\itshape before} the import itself, that won\textquotesingle{}t be re-\/ordered.
\end{DoxyEnumerate}\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9869}{}\doxysection{\texorpdfstring{When Not To Use It}{When Not To Use It}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9869}
If you don\textquotesingle{}t mind imports being sprinkled throughout, you may not want to enable this rule.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9870}{}\doxysection{\texorpdfstring{Further Reading}{Further Reading}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2first_autotoc_md9870}
