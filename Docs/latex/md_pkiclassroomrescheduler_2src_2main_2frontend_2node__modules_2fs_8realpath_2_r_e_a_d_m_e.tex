 A backwards-\/compatible fs.\+realpath for Node v6 and above

In Node v6, the Java\+Script implementation of fs.\+realpath was replaced with a faster (but less resilient) native implementation. That raises new and platform-\/specific errors and cannot handle long or excessively symlink-\/looping paths.

This module handles those cases by detecting the new errors and falling back to the Java\+Script implementation. On versions of Node prior to v6, it has no effect.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2fs_8realpath_2_r_e_a_d_m_e_autotoc_md13092}{}\doxysection{\texorpdfstring{USAGE}{USAGE}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2fs_8realpath_2_r_e_a_d_m_e_autotoc_md13092}
