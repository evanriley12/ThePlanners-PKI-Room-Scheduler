\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+SVG O\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}ptimizer is a Nodejs-\/based tool for optimizing SVG vector graphics files. \hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2svgo_2_r_e_a_d_m_e_autotoc_md24453}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2svgo_2_r_e_a_d_m_e_autotoc_md24453}
SVG files, especially those exported from various editors, usually contain a lot of redundant and useless information. This can include editor metadata, comments, hidden elements, default or non-\/optimal values and other stuff that can be safely removed or converted without affecting the SVG rendering result.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2svgo_2_r_e_a_d_m_e_autotoc_md24454}{}\doxysubsection{\texorpdfstring{What it can do}{What it can do}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2svgo_2_r_e_a_d_m_e_autotoc_md24454}
