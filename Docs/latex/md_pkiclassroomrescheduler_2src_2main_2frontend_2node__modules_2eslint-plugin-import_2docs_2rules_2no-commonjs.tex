Reports {\ttfamily require(\mbox{[}string\mbox{]})} function calls. Will not report if \texorpdfstring{$>$}{>}1 argument, or single argument is not a literal string.

Reports {\ttfamily module.\+exports} or {\ttfamily exports.\texorpdfstring{$\ast$}{*}}, also.

Intended for temporary use when migrating to pure ES6 modules.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2no-commonjs_autotoc_md9916}{}\doxysection{\texorpdfstring{Rule Details}{Rule Details}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2no-commonjs_autotoc_md9916}
This will be reported\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{var\ mod\ =\ require('./mod')}
\DoxyCodeLine{\ \ ,\ common\ =\ require('./common')}
\DoxyCodeLine{\ \ ,\ fs\ =\ require('fs')}
\DoxyCodeLine{\ \ ,\ whateverModule\ =\ require('./not-\/found')}
\DoxyCodeLine{}
\DoxyCodeLine{module.exports\ =\ \{\ a:\ "{}b"{}\ \}}
\DoxyCodeLine{exports.c\ =\ "{}d"{}}

\end{DoxyCode}

This is useful for things like ESLint rule modules, which must export a function as the module.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2no-commonjs_autotoc_md9920}{}\doxysection{\texorpdfstring{When Not To Use It}{When Not To Use It}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2eslint-plugin-import_2docs_2rules_2no-commonjs_autotoc_md9920}
