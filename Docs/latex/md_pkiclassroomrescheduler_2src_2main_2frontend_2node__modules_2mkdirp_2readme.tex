\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mkdirp_2readme_autotoc_md16962}{}\doxysubsection{\texorpdfstring{mkdirp(dir, opts, cb)}{mkdirp(dir, opts, cb)}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mkdirp_2readme_autotoc_md16962}
Create a new directory and any necessary subdirectories at {\ttfamily dir} with octal permission string {\ttfamily opts.\+mode}. If {\ttfamily opts} is a non-\/object, it will be treated as the {\ttfamily opts.\+mode}.

If {\ttfamily opts.\+mode} isn\textquotesingle{}t specified, it defaults to {\ttfamily 0777}.

{\ttfamily cb(err, made)} fires with the error or the first directory {\ttfamily made} that had to be created, if any.

You can optionally pass in an alternate {\ttfamily fs} implementation by passing in {\ttfamily opts.\+fs}. Your implementation should have {\ttfamily opts.\+fs.\+mkdir(path, mode, cb)} and {\ttfamily opts.\+fs.\+stat(path, cb)}.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mkdirp_2readme_autotoc_md16963}{}\doxysubsection{\texorpdfstring{mkdirp.\+sync(dir, opts)}{mkdirp.\+sync(dir, opts)}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2mkdirp_2readme_autotoc_md16963}
Synchronously create a new directory and any necessary subdirectories at {\ttfamily dir} with octal permission string {\ttfamily opts.\+mode}. If {\ttfamily opts} is a non-\/object, it will be treated as the {\ttfamily opts.\+mode}.

If {\ttfamily opts.\+mode} isn\textquotesingle{}t specified, it defaults to {\ttfamily 0777}.

Returns the first directory that had to be created, if any.

You can optionally pass in an alternate {\ttfamily fs} implementation by passing in {\ttfamily opts.\+fs}.
