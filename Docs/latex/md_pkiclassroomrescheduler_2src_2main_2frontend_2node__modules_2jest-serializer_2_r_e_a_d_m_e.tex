 Module for serializing and deserializing object into memory and disk. By default, the {\ttfamily v8} implementations are used, but if not present, it defaults to {\ttfamily JSON} implementation. Both serializers have the advantage of being able to serialize {\ttfamily Map}, {\ttfamily Set}, {\ttfamily undefined}, {\ttfamily NaN}, etc, although the JSON one does it through a replacer/reviver.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2jest-serializer_2_r_e_a_d_m_e_autotoc_md15439}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2jest-serializer_2_r_e_a_d_m_e_autotoc_md15439}
