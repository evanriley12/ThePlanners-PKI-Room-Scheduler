This plugin uses \href{https://cssnano.co}{\texttt{ cssnano}} to optimize and minify your CSS.

Just like \href{https://github.com/NMFR/optimize-css-assets-webpack-plugin}{\texttt{ optimize-\/css-\/assets-\/webpack-\/plugin}} but more accurate with source maps and assets using query string, allows caching and works in parallel mode.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7952}{}\doxysubsection{\texorpdfstring{Getting Started}{Getting Started}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7952}

And run {\ttfamily webpack} via your preferred method.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7953}{}\doxysubsection{\texorpdfstring{Note about source maps}{Note about source maps}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7953}
\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Works only with {\ttfamily source-\/map}, {\ttfamily inline-\/source-\/map}, {\ttfamily hidden-\/source-\/map} and {\ttfamily nosources-\/source-\/map} values for the \href{https://webpack.js.org/configuration/devtool/}{\texttt{ {\ttfamily devtool}}} option.\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

Why? Because CSS support only these source map types.

The plugin respect the \href{https://webpack.js.org/configuration/devtool/}{\texttt{ {\ttfamily devtool}}} and using the {\ttfamily Source\+Map\+Dev\+Tool\+Plugin} plugin. Using supported {\ttfamily devtool} values enable source map generation. Using {\ttfamily Source\+Map\+Dev\+Tool\+Plugin} with enabled the {\ttfamily columns} option enables source map generation.

Use source maps to map error message locations to modules (this slows down the compilation). If you use your own {\ttfamily minify} function please read the {\ttfamily minify} section for handling source maps correctly.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7954}{}\doxysubsection{\texorpdfstring{Options}{Options}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2css-minimizer-webpack-plugin_2_r_e_a_d_m_e_autotoc_md7954}
