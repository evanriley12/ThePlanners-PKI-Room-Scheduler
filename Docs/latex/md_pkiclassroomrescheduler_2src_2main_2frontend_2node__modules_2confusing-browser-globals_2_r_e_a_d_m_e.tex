 A curated list of browser globals that commonly cause confusion and are not recommended to use without an explicit {\ttfamily window.} qualifier.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2confusing-browser-globals_2_r_e_a_d_m_e_autotoc_md7627}{}\doxysection{\texorpdfstring{Motivation}{Motivation}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2confusing-browser-globals_2_r_e_a_d_m_e_autotoc_md7627}
Some global variables in browser are likely to be used by people without the intent of using them as globals, such as {\ttfamily status}, {\ttfamily name}, {\ttfamily event}, etc.

For example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{handleClick()\ \{\ //\ missing\ \`{}event`\ argument}
\DoxyCodeLine{\ \ this.setState(\{}
\DoxyCodeLine{\ \ \ \ text:\ event.target.value\ //\ uses\ the\ \`{}event`\ global:\ oops!}
\DoxyCodeLine{\ \ \});}
\DoxyCodeLine{\}}

\end{DoxyCode}


This package exports a list of globals that are often used by mistake. You can feed this list to a static analysis tool like ESLint to prevent their usage without an explicit {\ttfamily window.} qualifier.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2confusing-browser-globals_2_r_e_a_d_m_e_autotoc_md7628}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2confusing-browser-globals_2_r_e_a_d_m_e_autotoc_md7628}
