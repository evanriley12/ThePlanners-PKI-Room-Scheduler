By default the wrap is soft, meaning long words may extend past the column width. Setting this to {\ttfamily true} will make it hard wrap at the column width.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2_0disaacs_2cliui_2node__modules_2wrap-ansi_2readme_autotoc_md1347}{}\doxyparagraph{\texorpdfstring{word\+Wrap}{word\+Wrap}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2_0disaacs_2cliui_2node__modules_2wrap-ansi_2readme_autotoc_md1347}
Type\+: {\ttfamily boolean}\textbackslash{} Default\+: {\ttfamily true}

By default, an attempt is made to split words at spaces, ensuring that they don\textquotesingle{}t extend past the configured columns. If word\+Wrap is {\ttfamily false}, each column will instead be completely filled splitting words as necessary.\hypertarget{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2_0disaacs_2cliui_2node__modules_2wrap-ansi_2readme_autotoc_md1348}{}\doxyparagraph{\texorpdfstring{trim}{trim}}\label{md_pkiclassroomrescheduler_2src_2main_2frontend_2node__modules_2_0disaacs_2cliui_2node__modules_2wrap-ansi_2readme_autotoc_md1348}

Whitespace on all lines is removed by default.
